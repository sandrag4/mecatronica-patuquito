#  Proyecto de Mecatrónica – *Patuquito*

Este repositorio contiene todo el material necesario para el desarrollo del proyecto de la asignatura de Mecatrónica, cuyo objetivo es diseñar, montar y programar un **robot autónomo con patas**, capaz de **desplazarse y evitar obstáculos**.

El contenido está organizado para que cualquier usuario pueda **replicar**, **comprender** y **utilizar** el robot, ya sea con fines académicos o recreativos.

---

## Estructura del Repositorio

En las siguientes carpetas encontrarás todos los recursos del proyecto:

- [`codigo/`](./codigo/)  
  Contiene todos los códigos desarrollados durante el proyecto. Dentro de esta carpeta se incluyen:

  - [`codigo_robot/`](./codigo/codigo_robot/)  
    Código final utilizado por el robot para ejecutar los distintos modos de funcionamiento y comportamientos programados.

  - [`codigo_validaciones/`](./codigo/codigo_validaciones/)  
    Conjunto de códigos destinados a realizar las distintas pruebas y validaciones del robot, incluyendo test de motores, sensores, botones e inicialización.

- [`documentacion/`](./documentacion/)  
  Contiene los documentos generados durante el desarrollo del proyecto:  
  - **Informe de Autoevaluación** (diario de trabajo, problemas encontrados y soluciones aplicadas)  
  - **Informe Técnico** (documentación completa para replicar el proyecto en casa)  
  - **Manual del Usuario** (instrucciones para montar y utilizar el robot)  
  - **Manual de Mantenimiento** (indicaciones de cuidado y mantenimiento del sistema)

- [`enunciado/`](./enunciado/)  
  Carpeta que contiene el enunciado completo del proyecto.
